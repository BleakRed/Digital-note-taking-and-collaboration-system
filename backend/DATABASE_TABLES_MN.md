# Notion Клон Төсөл - Өгөгдлийн Сангийн Хүснэгтүүд

Энэ баримт нь төслийн Prisma schema (`prisma/schema.prisma`)-д тодорхойлогдсон бүх өгөгдлийн сангийн хүснэгтүүдийг (Prisma загварууд) орхиогүйгээр жагсаана.

---

## 1. Хэрэглэгч (User)
Хэрэглэгчийн бүртгэл болон нэвтрэх мэдээллийг хадгална.

### Талбарууд (Баганын нэр, ӨС нэр, Түлхүүр, Төрөл, Тайлбар)
| Баганын нэр       | ӨС нэр       | Түлхүүр | Төрөл    | Тайлбар |
|---------------------|-----------------|-----------|----------|----------|
| Хэрэглэгчийн ID    | id              | PK        | uuid     | цор ганц код |
| И-мэйл             | email           |           | varchar  | @unique, цаашид давтагдахгүй |
| Нууц үг            | password        |           | varchar  | Нууц үг |
| Нэр (эцэг)         | firstName       |           | varchar  | Сонголтой, эцэг нэр |
| Нэр (овог)         | lastName        |           | varchar  | Сонголтой, овог нэр |
| Нэр                 | name            |           | varchar  | Сонголтой, хэрэглэгчийн нэр |
| Хэрэглэгчийн нэр   | username        |           | varchar  | Сонголтой, @unique, цаашид давтагдахгүй |
| Аватар холбоос      | avatarUrl       |           | varchar  | Сонголтой, аватар зурагны холбоос |
| Баталгаажсан эсэх  | isVerified      |           | boolean  | анхдагч утга: худал |
| Баталгаажуулах токен | verificationToken |           | varchar  | Сонголтой, баталгаажуулах токен |
| Нууц үг сэргэх токен | resetToken   |           | varchar  | Сонголтой, нууц үг сэргэх токен |
| Токен дуусах хугацаа | resetTokenExpiry |           | datetime | Сонголтой, токен дуусах огноо |
| Үүсгэсэн огноо      | createdAt       |           | datetime | анхдагч утга: одоогийн огноо |
| Шинэчилсэн огноо    | updatedAt       |           | datetime | шинэчлэгдсэн огноо |

### Холбоосууд (Relations)
- `ownedWorkspaces`: Ажлын Талбар[] (холбоо: "Owner" - эзэмшигч)
- `memberships`: Ажлын Талбарын Гишүүн[]
- `createdPages`: Хуудас[]
- `uploadedFiles`: Файл[]
- `createdDrawings`: Зураг[] (холбоо: "DrawingAuthor" - зураг зохиогч)
- `chatMessages`: Чат Мессеж[] (холбоо: "ChatAuthor" - чат зохиогч)
- `kanbanCards`: Канбан Карт[] (холбоо: "Author" - зохиогч)
- `assignedCards`: Канбан Карт[] (холбоо: "CardAssignees" - карт хуваарилагдсан)

## 2. Ажлын Талбар (Workspace)
Ажлын талбарын мэдээлэл, хэрэглэгч эзэмшигчтэй.

### Талбарууд (Баганын нэр, ӨС нэр, Түлхүүр, Төрөл, Тайлбар)
| Баганын нэр   | ӨС нэр   | Түлхүүр | Төрөл    | Тайлбар |
|-----------------|------------|-----------|----------|----------|
| Ажлын талбарын ID | id         | PK        | uuid     | цор ганц код |
| Нэр              | name       |           | varchar  | Ажлын талбарын нэр |
| Эзэмшигчийн ID | ownerId    | FK        | uuid     | харьяалагдах хэрэглэгч |
| Үүсгэсэн огноо    | createdAt  |           | datetime | анхдагч утга: одоогийн огноо |
| Шинэчилсэн огноо  | updatedAt  |           | datetime | шинэчлэгдсэн огноо |

### Холбоосууд (Relations)
- `owner`: Хэрэглэгч (холбоо: "Owner", хамаарах: [ownerId], лавлах: [id])
- `members`: Ажлын Талбарын Гишүүн[]
- `pages`: Хуудас[]
- `files`: Файл[]
- `folders`: Хавтас[]
- `drawings`: Зураг[]
- `chatRooms`: Чат өрөө[]
- `kanbanBoards`: Канбан Самбар[]

### Гадаад Түлхүүрүүд (Foreign Keys)
- `ownerId` нь `Хэрэглэгч.id`-д хамаарна

---

## 3. Ажлын Талбарын Гишүүн (WorkspaceMember)
Хэрэглэгчийг ажлын талбартай холбож, эрх олгоно.

### Талбарууд (Баганын нэр, ӨС нэр, Түлхүүр, Төрөл, Тайлбар)
| Баганын нэр   | ӨС нэр   | Түлхүүр | Төрөл    | Тайлбар |
|--------------|------------|-----------|----------|----------|
| Гишүүний ID    | id           | PK        | uuid     | цор ганц код |
| Талбарын ID    | workspaceId  | FK        | uuid     | харьяалагдах ажлын талбар |
| Хэрэглэгчийн ID | userId       | FK        | uuid     | харьяалагдах хэрэглэгч |
| Эрх             | role         |           | varchar  | @default("MEMBER"), утгууд: OWNER, ADMIN, MEMBER |
| Элсэн огноо     | joinedAt     |           | datetime | анхдагч утга: одоогийн огноо |

### Цор Тэмдэглэл (Unique Constraints)
- `@@unique([workspaceId, userId])` (ажлын талбар болон хэрэглэгчийн хослол цор ганц)

### Холбоосууд (Relations)
- `workspace`: Ажлын Талбар (хамаарах: [workspaceId], лавлах: [id], устгахдаа: Cascade)
- `user`: Хэрэглэгч (хамаарах: [userId], лавлах: [id], устгахдаа: Cascade)

### Гадаад Түлхүүрүүд (Foreign Keys)
- `workspaceId` нь `Ажлын Талбар.id`-д хамаарна
- `userId` нь `Хэрэглэгч.id`-д хамаарна

## 4. Хуудас (Page)
Notion шиг хуудасуудыг хадгална, шаталсан бүтэц дэмждэг.

### Талбарууд (Баганын нэр, ӨС нэр, Түлхүүр, Төрөл, Тайлбар)
| Баганын нэр   | ӨС нэр   | Түлхүүр | Төрөл    | Тайлбар |
|--------------|------------|-----------|----------|----------|
| Хуудасны ID    | id           | PK        | uuid     | цор ганц код |
| Гарчиг          | title        |           | varchar  | хуудасны гарчиг |
| Агуулга         | content      |           | text     | @default(""), хуудасны агуулга |
| Талбарын ID    | workspaceId  | FK        | uuid     | харьяалагдах ажлын талбар |
| Зохиогчийн ID  | authorId     | FK        | uuid     | харьяалагдах хэрэглэгч |
| Эх хуудасны ID | parentId     | FK        | uuid     | шаталсан бүтэц (Page.id) |
| Үүсгэсэн огноо  | createdAt    |           | datetime | анхдагч утга: одоогийн огноо |
| Шинэчилсэн огноо | updatedAt    |           | datetime | шинэчлэгдсэн огноо |

### Холбоосууд (Relations)
- `workspace`: Ажлын Талбар (хамаарах: [workspaceId], лавлах: [id], устгахдаа: Cascade)
- `author`: Хэрэглэгч (хамаарах: [authorId], лавлах: [id], устгахдаа: Cascade)
- `parent`: Хуудас? (холбоо: "PageHierarchy", хамаарах: [parentId], лавлах: [id], устгахдаа: Cascade)
- `children`: Хуудас[] (холбоо: "PageHierarchy")

### Гадаад Түлхүүрүүд (Foreign Keys)
- `workspaceId` нь `Ажлын Талбар.id`-д хамаарна
- `authorId` нь `Хэрэглэгч.id`-д хамаарна
- `parentId` нь `Хуудас.id`-д хамаарна

| Тайлбар       | description  |           | text     | Тайлангийн тайлбар |
| Төлөв          | status       |           | varchar  | @default("OPEN"), утгууд: OPEN (Нээлт), RESOLVED (Шийдвэрлэгдсэн) |
| Мэдээлэгчийн ID | reporterId   | FK        | uuid     | харьяалагдах хэрэглэгч |
| Үүсгэсэн огноо  | createdAt    |           | datetime | анхдагч утга: одоогийн огноо |
| Шинэчилсэн огноо | updatedAt    |           | datetime | шинэчлэгдсэн огноо |

### Тэмдэглэл
- `reporterId` нь `Хэрэглэгч.id`-д хамаарна, гэхдээ Prisma schema-д албан ёсны холбоо тодорхойлогдоогүй байна.
